# Welcome to Minions Documents

We're excited to introduce **Minions Documents** — Versioned text documents, templates, and file attachments.

## What's Included

- **TypeScript SDK**: `npm install @minions-documents/sdk`
- **Python SDK**: `pip install minions-documents`
- **CLI**: `npm install -g @minions-documents/cli`

## Getting Started

Check out the [documentation](https://documents.minions.help) for a full guide.
