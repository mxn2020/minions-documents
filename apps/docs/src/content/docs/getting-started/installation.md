---
title: Installation
description: Install Minions Documents packages
---

## TypeScript / Node.js

```bash
npm install @minions-documents/sdk minions-sdk
```

## Python

```bash
pip install minions-documents
```

## CLI

```bash
npm install -g @minions-documents/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
