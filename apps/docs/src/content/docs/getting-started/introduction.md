# Minions Documents

Versioned text documents, templates, and file attachments

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-documents/sdk`) — Core library for Node.js
- **Python SDK** (`minions-documents`) — Python bindings
- **CLI Tool** (`@minions-documents/cli`) — Command-line interface
- **Web App** — Interactive playground at [documents.minions.wtf](https://documents.minions.wtf)
